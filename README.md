# 无波 —— 多源地震灾情信息平台

*Introduction in English please visit: [README-EN.md](README-EN.md)*

无波，意为不起波澜，以喻平安。

项目的核心任务是：该系统提供全国地震灾情信息管理功能，对不同来源的实时地震数据进行上传、解析、处理和分析、发布、可视化的全生命周期，并对外提供查询接口。

### 技术栈

- 消息队列：RabbitMQ
- 依赖管理：Maven
